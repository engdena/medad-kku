# Medad 🎓 — Strategic Student Portfolio for KKU

AI-powered Career Guidance Platform for King Khalid University
*From Graduate with a Degree… to Competent with Skills*

-----

## 📌 About

Medad is a bilingual AI-powered academic dashboard and student portfolio built for King Khalid University students. It bridges the gap between academic outcomes and Saudi labor market requirements by analyzing each student’s current courses and grades, then delivering a personalized career roadmap before graduation.

> Built at EDUTHON² — Innovation Hackathon in Education
> King Khalid University | July 2024

-----

## ✨ Key Features

- 📊 Academic Performance Analysis — Analyzes current course grades to identify strengths and weaknesses
- 🗺️ Career Roadmap — Personalized technical and soft skill recommendations for each student
- 🤖 AI Career Advisor (ChatBot) — An interactive chatbot that acts as your personal career consultant
- 🏆 Achievement Portfolio — Documents extracurricular activities and converts them into competitive advantages
- 🔗 Institutional Integration — Ready to connect with KKU’s Akademia / Nafath systems
- 🌐 Bilingual Support — Full Arabic / English support with RTL/LTR auto-switching

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18 or higher
- npm or bun

### Installation & Run
# 1. Clone the repository
git clone https://github.com/engdena/medad-kku.git

# 2. Navigate into the project folder
cd medad-kku

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev

Then open your browser at:
http://localhost:5173

### Additional Commands
# Build for production
npm run build

# Preview the production build
npm run preview

# Run tests
npm run test
